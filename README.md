Para correr el juego precisar descargar el archivo sh y el crontab.txt
Deberas meterlos en la misma carpeta

Requerimientos para correr el programa:
Situarse en el directorio en el cual se encuentra el archivo sh descargado (con cd)
Otorgate permisos para correr el programa ("chmod u=rwx winsurf.sh")
Ejecutar el programa con "./winsurf.sh"
